# Power Node Pattern

## Zweck
Erfassung elektrischer Größen und Berechnung abgeleiteter Messwerte.

## Charakteristik
- höhere Datenrate
- Kalibrierung relevant
- lokal berechnete Werte
- Debugging anspruchsvoll

## Typische Daten
- Spannung
- Strom
- Wirkleistung
- Frequenz

## Doku-Mindestanforderungen
- Messpfad
- Kalibrierung
- MQTT Topics
- Teststatus
- Debugging-Historie
