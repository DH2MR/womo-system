# Sensor Node Pattern

## Zweck
Erfassung von Umwelt-, Status- oder Zustandsdaten.

## Charakteristik
- publish-lastig
- geringe Rechenlast
- wenig lokale Logik

## Typische Daten
- Temperatur
- Luftfeuchte
- CO2
- Tankstand

## Doku-Mindestanforderungen
- Hardware
- Pinbelegung
- MQTT Topics
- Teststatus
